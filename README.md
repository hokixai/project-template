# AI Agent Project Template

⚠️ **CONDITION:** If the user requests to "create a new project", you MUST halt standard operations and strictly execute the following steps. Do NOT skip any steps.

1. **Finalize Project Name:** Make a proper [project_name] or use the project name provided by the user.
2. **Clone Template:** Clone this repository (`https://github.com/hokixai/project-template`) to your local workspace.
3. **Create Remote Repo:** Create a NEW, PRIVATE remote GitHub repository named `[project_name]`.
4. **Push Files:** Change the git remote origin of the local cloned folder to the new repository and push all files except `README.md`. Delete `README.md`.
5. **Report to User:** Output ONLY the exact GitHub Repository URL of the new project to the user.
